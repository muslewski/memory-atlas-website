# Contributing to memory-atlas-website

This repository is the **marketing one-pager** for [`memory-atlas`](https://github.com/muslewski/memory-atlas).

## Where to file what

| Topic | Repo |
|---|---|
| Tool bugs, CLI features, install failures | **[memory-atlas](https://github.com/muslewski/memory-atlas/issues/new/choose)** |
| Questions & ideas about the tool | **[memory-atlas Discussions](https://github.com/muslewski/memory-atlas/discussions)** |
| Website copy / layout / performance | Issues on **this** repo |

## Dev (website)

```bash
npm install
npm run dev      # Vite
npm run build
```

Please keep product install commands accurate (`npm install -g memory-atlas` / `npx …`).
