05 · quickstart

Raise the vault

Four clicks. Copy each line. Stamp only what you reviewed.

setup
$ npm install -g memory-atlas ⎘
#1 Install globally — bins: atlas · memory-atlas
$ atlas init ⎘
#2 Scaffold the vault in this repo
$ atlas check ⎘
#3 Verify claims + ledger
$ atlas stamp <slug> && atlas build ⎘
#4 Stamp reviewed zones, rebuild index
$ npx memory-atlas init ⎘
#1 No global install — run from the registry
$ npx memory-atlas check ⎘
#2 Verify without linking
$ npx memory-atlas build ⎘
#3 Rebuild map/index.md
$ git clone https://github.com/muslewski/memory-atlas.git ⎘
#1 Clone the product repo
$ cd memory-atlas && npm install && npm link ⎘
#2 Link the CLI from source
$ atlas init ⎘
#3 Same verbs as the published package
✓ Prefer memory-atlas if MongoDB Atlas CLI owns atlas on PATH.
